Fix code CNC có đài khoan

Mini tool dùng để fix lỗi code CNC khi xuất từ phần mềm Aspire cho máy CNC có đài khoan. Thường thời gian gọi đài khoan chậm, thay đổi dao trong đài khoan cũng chậm và 1 vài máy còn xuất hiện lỗi khi đài khoan làm việc spindle dao phay vẫn quay. Nạp toàn bộ code vào thư mục và fix nó trong 1 click với EWPs fix CNC.
